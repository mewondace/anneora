Quiet text field for newsletter signup / contact. Two shapes: hairline underline or glass capsule.

```jsx
<Input variant="capsule" type="email" placeholder="you@void.com" label="Join the mailing list" />
<Input variant="line" placeholder="Search tracks" />
```

Focus brightens the border and leaks a faint halo (capsule only). Keep labels short, uppercase, wide-tracked.
