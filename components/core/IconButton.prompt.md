Circular glass button for a single icon — play/pause, social links, menu toggles. Always pass `label`.

```jsx
<IconButton label="Play" variant="glow" size="lg">
  <svg width="20" height="20">…play triangle…</svg>
</IconButton>
```

Variants: `ghost` (hairline glass) and `glow` (filled bone — use for the main transport control). Sizes `sm 36 | md 44 | lg 56`.
